# TheraTrack System 🧩

TheraTrack is a web-based therapy management system designed to improve the scheduling, tracking, and communication of therapy sessions for children with Autism Spectrum Disorder (ASD).

---

## 📌 Background

Currently, therapy session management for children with autism is still widely handled using manual, paper-based methods such as appointment cards, printed invoices, and handwritten session reports.

These traditional methods often lead to several issues such as:
- Missed or poorly managed appointments  
- Lost or disorganised records  
- Slow and inefficient payment tracking  
- Communication gaps between parents, therapists, and administrators  

Such inefficiencies can disrupt consistent therapy scheduling, which is critical for children with ASD development progress.

---

## ❗ Problem Statement

The existing manual system creates multiple challenges:

- Appointment schedules are easily missed or not properly updated  
- Paper-based invoices make payment tracking inefficient  
- Therapy reports are difficult to store, retrieve, and manage  
- Lack of real-time communication between stakeholders  
- No centralized system for therapy session management  

Overall, the absence of an integrated system reduces efficiency and increases workload for carers and healthcare providers.

---

## 💡 Solution

TheraTrack System is developed to digitalize and centralize therapy management by providing:

- Centralized appointment scheduling system  
- Automated reminders for therapy sessions  
- Digital payment tracking and records  
- Online therapy session reporting system  
- Improved communication between parents, therapists, and administrators  
- Admin control panel for managing data efficiently  

---

## 🎯 Objective

To develop an efficient and centralized system that improves therapy management for children with autism by reducing manual work, improving coordination, and ensuring better tracking of therapy progress.

---

## 🚀 Key Features

- 📅 Appointment scheduling & management  
- 🔔 Automated reminders  
- 💳 Payment tracking system  
- 📝 Digital therapy reports submission  
- 👨‍👩‍👧 Communication between users (parent, therapist, admin)  
- 📊 Admin dashboard for monitoring activities  

---

## 🛠️ System Type

- Web-based system  
- Database-driven application  

---

## 📈 Expected Impact

TheraTrack System helps improve efficiency in autism therapy management by reducing paperwork, minimizing errors, and ensuring better coordination between all parties involved in a child’s care.
